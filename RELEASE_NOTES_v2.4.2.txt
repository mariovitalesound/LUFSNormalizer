-----------------

WORKFLOW
--------

1. Select input folder (WAV/AIFF files)
2. Select output folder
3. Choose preset or set custom LUFS
4. Click "Start Processing"
5. Results:
   - normalized/ → Files at target LUFS
   - needs_limiting/ → Files that need manual limiting first


HANDLING "NEEDS LIMITING" FILES
-------------------------------

When files exceed the peak ceiling:

1. Open the file in your DAW (Pro Tools, Reaper, etc.)
2. Insert a True Peak limiter (FabFilter Pro-L2, Waves L2, etc.)
3. Set ceiling to -1.0 dBTP
4. Render/Export
5. Run through LUFS Normalizer again

Alternatively in iZotope RX:
1. Open file
2. Loudness module → Set target LUFS and True Peak limit
3. Process

KEYBOARD SHORTCUTS
------------------
