WHY THIS MATTERS
----------------

**Professional Standards:**
- Pro Tools: Won't upsample without explicit warning
- iZotope RX: Warns against upsampling
- Broadcast facilities: Never upsample deliverables
- Mastering engineers: Upsample only for format compatibility, never for "quality"

**Technical Reality:**
- 48kHz file has frequency content up to 24kHz (Nyquist limit)
- "Upsampling" to 96kHz: Interpolates between existing samples
- No new frequency information above 24kHz
- Just inflates file size (2x larger) with no benefit
- Fake "high-resolution" audio

**User Intent:**
If user explicitly selects 96kHz output:
- Maybe they need it for DAW session compatibility
- Maybe they have specific workflow requirements
- Maybe they misunderstood what upsampling does

**In ALL cases:** They should know they're NOT getting real 96kHz, and the tool should NOT proceed without their explicit consent.
